# BlackJack Protocal

# Introduction
Developing Blackjack game, and using TCP to communicate between server and client.

## Install
```pip install -r requiments.txt```

## Run

### Server
```python Server.py```

### Client
```python Blackjack.py```
host: localhost
port: 8082
